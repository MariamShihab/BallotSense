# Review-only chunk candidates

These are verbatim excerpts prepared for human comparison against the private
source snapshots. They are deliberately not `SourceChunk` records: no item in
this directory is eligible for retrieval, embedding, or AI generation.

`needs_reviewer_comparison` means the source page was image-based and OCR was
used. The reviewer must compare it to the cited page before approval.

`prop-36-2024.json` is expected to remain pending until the reviewer compares
each page-level extraction candidate against the official 2024 voter guide PDF.
It must not be promoted or embedded before explicit approval.
