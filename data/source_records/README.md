# Pending source records

This directory records the immutable, private Cloud Storage snapshot that a
reviewer must approve or reject. Records with `review_status: pending` are not
available to retrieval or generation. Raw PDFs remain only in the private
artifact store; this repository holds provenance metadata and review-only text
excerpts.

The active pending target is `prop-36-2024.json`, backed by the official
California Secretary of State November 2024 Voter Information Guide master PDF
manifest.
